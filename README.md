# I Wanna Eat Out

SSPU automatic applying for leaving school

## 我要出去吃饭

这是一个用 Python 和 Selenium 实现的自动申请出校工具，它可以帮助你快速出校吃饭。

通过内置的定时器，可以实现让你在任意时间出校。

## 功能特性

* 单次运行
* 定时运行
* API

## 浏览器支持

* Google Chrome
* Firefox *(Not tested)*
* Microsoft Edge *(Not tested)*
